Dealing with FEC Donation Data

Previous Next

Project Description

  • Completed in CS:2420: Analyzing Data for Informatics
  • Implemented in a Jupyter Notebook: link here
  • Used donation data from the Federal Election Commission
    • Iowa City residents’ donations to House candidates in 2023-2024
  • Created pandas dataframes based on the dataset
  • Mapped party information to each candidate
  • Streamlined types of employment
    • Ex. different types of teachers all became “Teacher”
  • Displayed data with Matplotlib

What I Learned

  • How to create dataframes in Python from publicly available data
  • How to map additional information onto a dataframe
  • How to visualize a streamlined dataset in Matplotlib
